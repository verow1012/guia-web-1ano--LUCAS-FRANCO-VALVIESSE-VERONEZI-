Domine as tecnologias fundamentais da web moderna

Começar

HTML5

A linguagem de marcação que estrutura todo o conteúdo na web.

<!DOCTYPE html> <html lang="pt-BR"> <head> <title>Título</title> </head> <body> <h1>Olá Mundo!</h1> </body> </html>

Use para estruturar documentos web e criar interfaces semânticas.

CSS3

A linguagem de estilo que define a aparência e o layout dos elementos.

.button { background: #2563EB; color: white; padding: 1rem 2rem; border-radius: 4px; transition: transform 0.2s; } .button:hover { transform: scale(1.05); }

Use para estilizar elementos HTML e criar layouts responsivos.

JavaScript

A linguagem de programação que adiciona interatividade às páginas.

const button = document.querySelector('.button'); button.addEventListener('click', () => { console.log('Botão clicado!'); button.classList.add('active'); });

Use para criar interações dinâmicas e manipular o DOM.
